# Sobre o projeto

Aplicação desenvolvida como desafio do Ignite.

# Tecnologias

- React
- Next
- Chakra UI

# Como rodar?

```bash
# clone este repositorio
git clone git@github.com:joaovictor3g/coffee-delivery.git

# entre no diretório
cd coffee-delivery

# instale as dependências
yarn || npm i

# Rode o fake server
yarn server

# Rode o projeto
yarn dev
```
